Introduccion a CSS
Opacidad
background-color: Especifica el color de fondo de un elemento.
background-image: Define una imagen de fondo para un elemento.
background-repeat: Controla si la imagen de fondo se repite o no.
repeat: Se repite tanto horizontal como verticalmente.
no-repeat: No se repite la imagen.
repeat-x: Se repite solo en el eje horizontal.
repeat-y: Se repite solo en el eje vertical.
background-attachment: Define si la imagen de fondo se desplaza o no.
scroll: La imagen se desplaza con el contenido.
fixed: La imagen permanece fija cuando se desplaza la página.
local: La imagen se desplaza con el contenido del elemento.
background-position: Especifica la posición de la imagen de fondo.
Se puede usar valores como top, center, bottom, left, right o valores específicos en píxeles o porcentajes.
