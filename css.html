Introduccion a CSS
Esctructura de CSS
  • Selector: Es el elemento al que se aplica el estilo. Ejemplo: p, .clase, #id.
  • Propiedad: Es lo que queremos modificar del elemento. Ejemplo: color, font-size.
  • Valor: Es el valor que se asigna a la propiedad. Ejemplo: red, 16px
El modelo de cajas describe cómo se organiza el espacio de un elemento HTML.
Contenido: Lo que contiene el elemento.
Relleno (Padding): Espacio entre el contenido y el borde.
Borde (Border): El borde alrededor del contenido.
Margen (Margin): Espacio fuera del borde.
Elementos de bloque: Ocupan todo el ancho disponible, comienzan en una nueva línea. Ejemplo: div, p.

Elementos de línea: Solo ocupan el espacio necesario, no interrumpen el flujo de contenido. Ejemplo: span, a.
